| Sanitizer                      | Description & Parameters                                                                                                                                                                                                                    |
| ------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `blacklist(str, options)`      | Replace characters or words from a "blacklist" with a replacement character. By default it removes the matched characters (replaces with ''). - `str`: `string` — The string to be sanitized. - `options`: `object` — Sanitization options. |
| `escapeHTML(str)`              | Convert special HTML characters ('&', '<', '>', '"', "'") to their corresponding HTML entities. - `str`: `string` — The string to be escaped.                                                                                               |
| `lTrim(str, options)`          | Remove whitespace (or custom characters) from the beginning (left) of the string. - `str`: `string` — The string to be "cleaned". - `options`: `object` — Options (e.g., { chars: '\_-' })                                                  |
| `normalizeEmail(str, options)` | Normalize an email address into a canonical form. - `str`: `string` — The email string to normalize. - `options`: `object` — Sanitization options.                                                                                          |
| `rTrim(str, options)`          | Remove whitespace (or custom characters) from the end (right) of the string. - `str`: `string` — The string to be trimmed. - `options`: `object` — Options (e.g. { chars: '\_-' }).                                                         |
| `stripTags(str, options)`      | Remove HTML/XML tags from a string, optionally preserving a whitelist of tags. - `str`: `string` — The string to be cleaned. - `options`: `object` — Sanitization options.                                                                  |
| `toSlug(str, options)`         | Convert a string into a URL-friendly "slug". - `str`: `string` — The string to convert. - `options`: `object` — Sanitization options.                                                                                                       |
| `trim(str, options)`           | Remove whitespace (or custom characters) from both ends of the string. - `str`: `string` — The string to be trimmed. - `options`: `object` — Options (e.g. { chars: '\_-' }).                                                               |
| `unescapeHTML(str)`            | Convert basic HTML entities ('&amp;', '&lt;', '&gt;', '&quot;', '&#x27;') into their corresponding characters (''', '<', '>', '"', "'"). - `str`: `string` — The string to be unescaped.                                                    |
| `whitelist(str, options)`      | Remove from the string all characters or words that are NOT in the "whitelist". - `str`: `string` — The string to be cleaned. - `options`: `object` — Sanitization options.                                                                 |
